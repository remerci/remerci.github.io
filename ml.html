    计出更好的预测机制
  • 校准不确定度估计也有助于基于协变量（基于内容的过滤）或过去的用户点击行为（协同
    过滤）来做出预测

  • 线性空间中的基是非奇异(向量线性无关)方阵中的向量, 坐标系也是矩阵；
  • MA=B 表明坐标系 M 下 A 变换等同于坐标系 I 下的 B 变换, 运动的相对性；
  • 非方阵, 线性变换作为一种映射, 核与像等概念；
  • 行列式是组成 M 的各向量按照平行四边法则搭成一个 n 维立方体的体积；
  • 数学分析的观念: 一个对象可以表达为无穷多个合理选择的对象的线性和；
  • 齐民友的<重温微积分> -> 高等数学是研究运动的数学

1.2.5.5 张量

张量的物理本质是多重方向性, 数学上的对应就是线性变换。所有的变换、基底、坐标都只是用来描述这个多重方向性的
工具。相应地，标量就是不具有方向性的物理量，矢量就是具有一重方向性的物理量。

流形
流形（manifold)是局部具有欧几里得空间性质的空间，是欧几里得空间中的曲
线、曲面等概念的推广。
切空间
切空间是在某一点所有的切向量组成的线性空间。向量（切向量）存在多种定
义。直观的讲，如果所研究的流形是一个三维空间中的曲面，则在每一点的切向量，就
是和该曲面相切的向量，切空间就是和该曲面相切的平面。
二次型
二次型，quadratic form。 线性代数 的重要内容之一，它起源于几何学中二次
曲线方程和二次曲面方程化为标准形问题的研究。二次型理论与 域 的特征有关。
对偶空间
对偶空间构造是 行向量(1×n)与列向量(n×1)的关系的抽象化。这个结构能够
在无限维度空间进行并为测度,分布及希尔伯特空间提供重要的观点。对偶空间的应用
是泛函分析理论的一特征。 傅立叶变换亦内蕴对偶空间的概念。

  • 与其它关系
    递推

    每个阶段的最优状态都是由上一个阶段的最优状态得到的
    搜索
    每个阶段的最优状态是由之前所有阶段的状态的组合得到的

    每个阶段的最优状态可以从之前某个阶段的某个或某些状态直接得到而不管之前这个状态是如何得到的

  • 应用需求：时效，相关，文档（精度，类型，单位，来源） / 抽样偏倚
  • 预处理
    • 聚集
    • 抽样 / 随机, 分层, 容量, 渐近
    • 降维 / PCA, 奇异值分解 (SVD, singular value decomposition)
    • 特征
    • 离散 / 熵
    • 变换 / 归一, 保序
  • 度量相似性
    • 闵可夫斯基距离
    • Jaccard 系数
    • 余弦系数
    • Pearson 系数
    • Bregman 散度 / 凸函数, 梯度, 内积
