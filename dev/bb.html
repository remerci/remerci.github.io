1.1 注册激活码

以前看了 BerryAble 的教程后，一直是使用 OLLYDBG 从内存中查找注册码，查找 BerryBuzz 之类
的软件的注册码还是很方便，基本上可以直接找到。但找 AA 的注册码就成了很痛苦的一件
事，通常重复好几次还是找不出正确的注册码。

后来看到有人说用 winhex，就去网上下载了一个，试用了几次，感觉比 OLLYDBG 要方便很多。
还可以用它来查找一些 Windows 应用程序的注册码。比如 WinImage，TraceMe unicode 版（网
上看到的，自己没试）。

Winhex 的使用方法也很简单，比如要查找 AA 的注册码：

把模拟器的 pin 码改成跟你手机一样，运行 Blackberry 模拟器（我用的是 BlackBerry JDE
4.6.1，下载的 4.5 的有问题，不能用），安装好 AA 软件，然后随便输入一个注册码，比如

  1. 确定之后系统会提示“Invalid Serial Number”，不要关闭这个对话框，打开

Winhex，按下 ALT+F9，在列表中找到模拟器（Fledge），点击前面的+号展开，选择
“Entire Memory"，打开后，按下 CTRL+F 搜索字符串“Invalid Serial Number”或者之前输入
的注册码 49520，在找到的结果附近看看，如果没有注册码，就按 F3 继续往下找，BerryBuzz
可以在第一次出现 49520 的附近找到一个 5 位数的号码，就是最后的注册码，AA 可能要往下搜
2、3 次，才会出现。

1.2 barry

linux 平台上类似于桌面管理器功能的软件，现在主要功能是：

  • 给电池充电
  • 备份，同步
  • 拨号上网
  • javaloader（cvs 版本）

1.3 作为 modem 带电脑上网

1.3.1 拨号

只需要两个文件，其中 cmwap 放在/etc/ppp/peers/下，cmwap.chat 放在~/bin 下。
而后用以下命令即可连接
<example>
sudo pppd call cmwap
</example>
将 cmwap.chat 中的 cmwap 改成 cmnet 就可以 cment 上网

1.3.2 firefox 设置

  • 修改代理：10.0.0.172:80
  • 修改浏览器标识

在地址栏输入 about:config，
然后查找 general.useragent`，把 value 改成 Netscape/4.8 就可以了。

  • 加载支持 wap 页面查看和减小流量的插件

1.4 与电脑的文件传输

因为没有卡，靠手机内存与电脑交互，所以一般的方法是通过 win 上桌面管理器来。

linux 上有个 libmtp 在软件可以用，但现在需要加 patch 才可以识别 blackberry storm, 恐怕
还需要在该 patch 基础上再改。

linux 上蓝牙文件传输很不方便，用 gnome-obex-server 命令可以用电脑接收文件，但 10 次尝
试只有 1、2 次成功。电脑发送文件还没成功过。

1.5 导入 service book

先在 options - advanced options - service book 选项里按
Alt+S B E B，然后再导入，否则导入无效。
