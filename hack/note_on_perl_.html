1 perl 特别之处

  • 字符串： .和 x 操作符和 chomp 函数
  • $#和-1 对应数组最后一个元素下标
  • 列表： ..操作符和 qw 函数
  • foreach 循环控制变量、@_作用域，变量初值
  • 子程序返回值为 the last expression evaluated
  • $_: 循环控制变量，print, <STDIN>, chomp

2 数据类型：列表（数组）和哈希变量（3，5 章）

2.1 对于列表变量

特殊用法有：
<example>
($a,$b)=($b,$a) #交换两个变量的值
</example>

特殊的操作有：

  1. push, pop, shift, unshift: 右边加入，删除一个元素；左边删除插入一个元素
  2. reverse, sort: 顺序颠倒；按字符串排序
  3. @fred[2,3]: slice，取出列表中的一段内容

2.2 对于哈希变量

特殊用法有：
<example>
@fred_list=%fred
%binary=@fred_list #哈希按列表变量存取，%fred 和%binary 内容一样

@score{"fred", "barney", "dino"} = (205, 195, 30); #hash slice
</example>

特殊操作有：

  1. reverse: key 和 value 反过来
  2. keys values: 得到全部为 key 的 list 和全部为值的 list
  3. each(%fred); delete $fred("aaa"): 顺序取出一个 hash 对到列表变量；删除一个 hash 对

3 控制和基本 I/O（4，6 章）

if/unless, while/until 分别有相反的意思

<example>
foreach $i (@list){
$i*=3; #可以改变@list 内的值；$i 的循环后保持原样
}
</example>

print chomp foreach <STDIN>默认用$_变量

<src lang="perl">
#! /usr/bin/perl
while(<>){
print;
} #输入的内容来自@ARGV（默认为命令行参数，可变）
</src>
