1 区分 (第七、九)

  • () subshell
  • {} no name function
  • $@ $*都表示所有位置参数，在双引号内有别

<src lang="shell-script">
my.sh p1 "p2 p3" p4
echo "$@" "p1" "p2 p3" "p4" 三個不同的詞段(word)
echo "$*" "p1 p2 p3 p4" 一整串單一的詞段
</src>

3 单双引号

单引号关闭更多 meta(特殊字符)含义。

  • shell script 中的 awk、sed 等命令的参数如何正确被识别：单引号关闭 shell meta.
  • awk 中又用到 shell script 中的变量呢：关闭部分.

<src lang=shell-script>
A=0
awk "{print \\[A}" 1.txt awk \{print\ \\]A\} 1.txt
awk '{print $'$A'}' 1.txt
awk '{print $'"$A"'}' 1.txt
</src>
