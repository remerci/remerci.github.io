1 想实现类似 office 2007 界面效果(Ribbon?)

  • 依托容器控件, 例子迅雷界面. 用 split, panel, tablelayout 组合
  • 用 qt, wpf 1, gtk+等界面框架
  • 用 DotNetBar 或 DevExpress 收费控件
  • 用 irisskin2 皮肤风格

最终选择 DotNetBar.
WPF 主要为了随意控件 windows 控件的位置，显示比较炫的界面。你如果想做 UI 设计师。 可以深入 WPF 专做这个。 什么渐变, 谈化, 透明什么的。

2 程序要求

  • 菜单右侧显示 炮号, 测试时间
  • tree 为测试项目, 不同项目显示多个可选曲线, 右侧为总线曲线中当前点数值
  • 背景黑色, 点状 grid, 40ms 总长 30s 共 750 个点. 放大最大 1 ms 1 个点.
  • 趋势测试, 3 个曲线连接.

3 笔记

3.1 混合模式程序集是针对“v2.0.50727”版的运行时生成的

添加新项，应用程序配置。实际就是在程序源码目录下添加了 app.config 文件。
在 app.config 中的 configuration 节内添加子节 Startup，详细如下：

<startup useLegacyV2RuntimeActivationPolicy="true">
    <supportedRuntime version="v4.0" sku = ".NETFramework,Version=v4.0"/>
    <supportedRuntime version="v2.0.50727"/>
</startup>

3.4 皮肤控制

  • dotNetBarManager 属性可设置 color scheme，但找到颜色配置文件, dcs 后缀。
  • ribbon 风格控件可在属性里直接改。
  • form 继承 Office2007RibbonForm, 在 form_load 函数里用下列语句:
RibbonPredefinedColorSchemes.ChangeOffice2007ColorTable(this, DevComponents.DotNetBar.Rendering.eOffice2007ColorScheme.Black);
