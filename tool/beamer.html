实验文档 图形

最近的报告（关于传感网体系结构和可能应用）是用 beamer 做的，参考了网上的一个源文件。这里是一些学习中的收获。

1 一般

1.1 中文 pdf 书签, 中文拷贝

中文书签乱码原因在于 pdf 只认 unicode 编码的书签. 故
gbk 编码, 用 gbk2uni 处理.out 文件即可.
utf8 编码应该可直接生成无乱码书签, 但 beamer.cls 中并未加入对 CJKutf8 宏包的支持.
按 scrapbook 中的方法改 beamer.cls 文件即可.

用 ccmap 宏包即可用 pdflatex 生成支持中文复制粘贴的文件了.

1.2 加入中文支持

导言区的设置
<example>
\documentclass[cjk]{beamer} % 使用 Beamer 包
\usepackage{CJKutf8} % 中文环境
</example>

在\begin{document}后，正文前加入\begin{CJK}{UTF8}{song}。
正文后\end{document}前加入\end{CJK}即可。

1.3 overlay 效果

\item 后<2>，<3->分别表示该项在第二片出现，在第三片以后出现。
简单的可直接在\itemize 后跟[<+->]选项。

相应的高亮则为
<example>
<2| alert@2>，<3-| alert@3>，[<+-| alert@+>]
</example>

1.4 插入图片

可以直接用\includegraphics 命令插入图片，也可用下面的图形环境，以便加入图注。
<example>

\begin{figure} \centering \includegraphics[height=6cm]{mesh_sensor.jpg} \end{figure}

</example>

1.5 胶片结构

\itemize 命令可以嵌套。还有\enumerate，和\description。
双栏结构如下：
<example>

\begin{columns} \column{3cm} \column{7cm} \end{columns}

</example>

2 技巧

2.1 致谢的大字体 thank you

<example>
\frame{

\begin{center} {\Huge \emph{\textcolor[rgb]{0.5,0,1.0}{Thank you!}}} \end{center}

}
</example>

2.2 add notes

导言区添加命令\setbeameroption{show notes no second screen}，控制是否显示，是否只显示 notes。胶片中用\note[item]{}命令来加入 notes。

2.3 改变整个幻灯片的字号

<example>
\documentclass[cjk,12pt]{beamer} % 使用 Beamer 包
</example>
默认为 11pt。

2.4 实现下面作者信息的效果


<example>
\author[博士生：赵鑫]{

\begin{tabular}{ll} 博士生： & \href{mailto:xinzhao\_remerci@nudt.edu.cn}{赵鑫}\\ 导师：& 周兴铭\quad 教授\\ & 王晓东\quad 副研究员 \end{tabular}

}
</example>
用表格环境实现，其中方括号内是简写，用于导航栏页底信息栏的显示。

3 关于 beamer 中的图形和动画

beamer 的缺陷主要在动画(如报文传输), 和确定引用图像的位置不方便.

还没找到合适的动画解决方案, beamer 中是用多页面实现类动画效果, 文件会很大, 也占
资源.

图形工具支持 metapost 和 pgf, pstricks 好像也可以(见 beamer-guide).

beamer 对图形的支持还是有一些的. 如区域放大某图, 用 mask 功能获得透明背景的图形.

3.1 beamer 图形位置

pgfputat 可以将图形放到任意位置. 但不容易快速找到合适位置. 位置范围是 (0cm,7cm)
× (11cm,1cm)

下载网页上技巧, 见下面代码:
<src lang="latex">
\setlength{\unitlength}{\textwidth} % measure in textwidths

\frame{
Hello\par
\framebox{ % just so you can see where the "picture" is

\begin{picture}(0.1,0.1) \put(0.2,0.05){\includegraphics[width=0.5\textwidth]{png}} \end{picture}

}\par
Goodbye
}
</src>
