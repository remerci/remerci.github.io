3.6 other

  • LyX: 一个所见即所得(WYSIWYG)的 LaTeX 编辑器
  • 数值计算程序：SciLab。这个程序基本上可以代替 Matlab。
  • 代数计算程序：MAXIMA。这个程序基于世界上最老的计算机代数系统之一: 由美国能源部(DOE)发行的 MIT Macsyma 系统。它是用 Common Lisp 实现的。很多现在的符号计算程序比如 Maple 都从 MAXIMA 身上学到很多东西。它现在经过 DOE 批准以 GPL 发行，永远是一个自由软件。
  • email 处理：Mutt + Postfix + fetchmail
  • 上 BBS：rxvt(或任何一种终端) + telnet + chatbot(helloooo 机器人的程序)
  • DrGeo: A Math Teaching Toy that Physics Teachers Will Also Love.

-—

  • dsh,cssh: 分布式 shell
  • puppet: 多机配置工具，也可用于单机配置备份
  • retty: 抓回后台程序
  • nas: network audio system
  • Synergy：通过网络共享键盘鼠标

-—

  • PovRay #渲染器；都说很吊，所以不能不提，感觉编程人员用她更好。
  • Qcad #CAD 图纸设计
  • Eric: 功能完备的 Python IDE
  • XML Copy Editor: XML 编辑器 http://xml-copy-editor.sourceforge.net/
  • CSSED: Linux 下 Web 开发者的 CSS 编辑利器
  • 网页制作: nvu, QuantaPlus, Aptana

-—

  • nuke: 视觉效果合成
  • Blender #三维制作;
  • soundKonverter #音频转换
  • Mainactor #视频编辑；算是 linux 上最强的吧
  • Mencoder/Ffmpeg #视频转换；这两个是必装的。
  • pyvideoconv 一个基于 python 和 gtk 的 mencoder 前端,制作 mp4 3gp
  • byzanz: 录制屏幕的 GIF 动画的小工具
  • Xvidcap #屏录软体；屏幕录制，支持多种格式，感觉比 WIN 下的多数此种软件都强，配合 Gromit，可以做教学视频。
  • iTalc（Intelligent Teaching and Learning with Computers）电子教室管理软件

  • Rawstudio: http://rawstudio.org/ 一款用来读取并处理 RAW 图片的开源程序，它支持大多数的数码相机。
  • PCSX2: PlayStation 2 游戏模拟器
