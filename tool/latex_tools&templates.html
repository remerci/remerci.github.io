3 auctex 的辅助功能

auctex 具有辅助编译、编辑的功能，很方便和强大。
编译功能包括 xdvi 反向定位（ctrl＋鼠标左键），比较简单。

常用辅助编辑的功能：

  • C-c C-m: 输入 latex 命令
  • C-c C-e: 输入 latex 环境
  • M-return: 输入 item
  • C-c C-f C-b: 输入\textbf{}
  • C-c {: 一对大括号
  • C-c ]: 关闭当前环境
  • C-c C-q C-r: 排列当前区域
  • C-c = 生成目录
  • C-c ;: 注释当前区域，前面加 C-u 取消注释
  • C-c %: 注释一段
  • C-c &: 跳到交叉索引处
  • C-c ~: 进入数学模式；`加字母希腊字母
  • C-c C-r: 区域编译，加快测试速度
  • M-C-a M-C-e: 跳到当前环境头和尾

3.1 C-c C-c 找不到 latex 命令

在文件/etc/environment 的 PATH 环境变量里添加 latex 的 bin 路径。

3.2 部分编译 CJK

默认部分编译时的_region.tex 文件不会自动添加 cjk 支持，需要手动添加才能进行编译。
自动编译的方法网上找到，改动有.emacs 文件（我的在.emacs.d/conf/auctex.el）中一些
设置，及 auctex/tex-buf.el 文件中两处修改。

5 latex-suite 与 vim

  1. 修改 ftpplugin/latex-suite/texrc 文件，如 defaultTargetFormat 改成 pdf.
  2. F5 可加入\begin{} \end{}, F7 可加入命令和相应参数
  3. ([{"自动转换成\left, ``等并匹配
  4. 三字母快捷键，如 EMA 自动替换为 math 环境
  5. 自动完成添加<++>定位符

6.1.1 通用

中文缩进，间距设置
<example>
\usepackage{CJK,indentfirst} %CJK 宏包

\CJKindent %段首缩进
\setlength\parindent{2em}%首行缩进 2 字

\CJKtilde %中英文间空格

\setlength\parskip{5pt}%段间距离设为 5pt
\renewcommand\baselinestretch{1.2}\normalsize%设置行距为 1.2 倍
\showthe\baselineskip %编译时得到\baselineskip 值
</example>

中文标点问题
