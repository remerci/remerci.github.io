fcitx 使用

越来越觉得仔细看一下程序的文档会发现许多有用的功能，如 man 中的 EXAMPLES 就可以给我很多
使用提示。费话不多说，说一下 fcitx 五笔输入发现的技巧。fcitx 的拼音输入功能也很多，
现在用五笔，就先不管了，以后用的时候再说。

  • 快速输入：五笔输入好象还没有添加用户词库的功能，当然可以通过自己改码表添加，不
    过麻烦多了。这个可以做补充，还可以添加常用长词的快捷输入，具体在
    .fcitx/QuickPhrase.mb，是个文本文件。用“分号+wsn”输入“无线传感网”还是挺爽的。
  • 临时拼音：z 开头，同时会显示字的代码，五笔输入显示的代码常是英文，五笔拼音显示
    的更正常些，并且可以直接打拼音，但不支持模糊键，zzzz 符号输入（它用符号的五笔
    码输入符号）等。
  • 反查拼音：不认识的字可以先用五笔打出来再反查其读音，强！
  • 2、3 字选择：左 SHIFT 选第 2 个候选字，右 SHIFT 选第三个，比直接按 2、3 快些，习惯了应
    该可以提高效率。

不太爽的地方 :: 快捷键设置比较死板，且有的设置不起作用。如联想功能不管怎么改它的
快捷键都是 Ctrl_L，切换输入法的 Ctrl+space 也基本是只此一家可用，不
过习惯了在 emacs 中用 Ctrl+@来 set mark也就没什么了。
