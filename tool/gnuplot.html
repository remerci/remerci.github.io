    • 0.1 一个小技巧

    以下 term 设置可以在不需要中文和公式的情况下输出 pdf，不生成中间文件
    <example>
    set term postscript
    set output "| ps2pdf - buff.pdf"
    </example>

    1 柱状图

    好象需要 gnuplot 的版本较高才支持. 旧版本有个 bargraph.pl 可用, 但需要
    fig2dev (transfig) 的支持. 具体内容见/windata/doc.
    <example>
    set style fill pattern 3 border -1
    set style histogram clustered gap 1 title offset character 0, 0, 0
    #set datafile missing '-'
    set style data histograms
    set boxwidth 0.9 absolute

    set xtics ("Normal" 0, "GKMAC" 1, "GBA" 2, "TinyECC" 3, "Hybrid" 4)
    plot 'size1.data' using 2 ti col title "ROM", '' u 3 ti col title "RAM"
    </example>

    size1.data 内容为
    <example>
    ROM RAM
    0 13456 862
    1 17740 912
    2 20358 1150
    3 29346 2450
    4 35625 3070
    </example>

    2 确定数据的输出环境

    <example>
    set terminal(term)
    </example>
    默认是 x11，直接画图，可以设为 postscript enhanced(post enh)，生产 eps 文件，设为 latex，直接生成 tex 文件。

    tex 文件直接在图形环境中用 input 引入，其线和点的形状和 eps 中的不同，网格也没能保留，因此在文章中还是采用了 eps 图形。

    3 图例方框的 bug?

    在图形中使用了网格，这时图例所在由空白方框覆盖了网格，不太好看。使用方框包含图例是一个比较好的选择。

    但在我的图例中包含了特殊的希腊字母 mu，即 title "{/Symbol m}TESLA"。这时发现方框的左边线流出了大量的空白，应该是为双引号内字符串变换前留出来的。为了将其变得合适，曾想试着改 eps 文件，看不懂。最后通过如下语句搞定：
    <example>
    set key left width -7 height 0.5 box
    </example>
    关键是 width 选项后的-7。

    这个问题在 term 设定为 pslatex, epslatex 等与 latex 相关的 term 时十分突出，无法用上面的办法解决，gnuplot 文档里建议用命令 set key left Left reverse 来解决，但对我例子不起作用。这是 gnuplot 目前对我来说的主要缺陷，最终的解决办法是 set key off*，然后 *set label '{\footnotesize $μ$TESLA}' at 4,6 ，即不使用 key 而直接在数据线上标出 label。注意在下一张图开始前要先 *unset label*，否则上面图的 label 会带下来。

    4 epslatex term 与 latex

    epslatex term 会输出用于描绘图形的 eps 文件和用于写文字信息的同名 tex 文件，图形大小可以在 gnuplot 脚本中用 set size 0.6,0.6 这样的命令设定，这样不会影响文字信息（label, key 等）的大小。另一个好处是文字可以用 latex 方式写，gnuplot 脚本例子：
    <example>
    #!/usr/bin/gnuplot -persist # -persist means 图形不随 gnuplot 关闭而关闭，用于 x11 term

    set term epslatex blacktext (or color) # default is color, tex file needs usepackage color.sty
    set output buff.tex
    set ylabel '{\small \shortstack{key authentication\\ time(ms)}}'

    #plot your figure

    set output # close output file
    set term pop # set term back to popular term
    </example>

    而在 latex 文件中用命令 \input{buff} 来引入该图形。

    5 最新关于图例和 term

    基于下面两条认识：

    • gnuplot 中 set size 命令只影响图形大小，文字(label, key)不受影响；而在 latex 文件中

    includegraphics 命令 width 等选项将同时影响图形和文字大小。

    • set key 中 width 和 height 选项调整文字部分的高宽，从而可调整 key 与边框的关系；Left
      和 Right 调整 key 中文字对齐方式，默认是 Right。

    当 key 中有公式时 (如 \(\mu\)), 由于 latex 相关 term 的 key bug, 使用 eps term.
    其它情况可用 epslatex term, 包含中文时可用 xelatex 中的 truetype 字体是这一方案
    的优势.

    • gnuplot 产生的图形，先用 set size 大致调整到文章中所需大小，再在 latex 文件中用

    width=.48\textwidth 选项或 \resizebox 命令调整，这样图中文字不会太小。

    • 用 width 调整文字与边框关系，特殊字符如$μ$可用 psfrag 或 overpic 宏包在 latex 文件中搞定。
