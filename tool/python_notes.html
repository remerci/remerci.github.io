  • 函数参数:
    • 默认参数设计为不可变
    • 可变参数, 定义用 *p, 调用可传入*list, *tuple, 多个不定数量参数 -> tuple
    • 关键字参数 **kw -> dict
    • 参数定义的顺序必须是：必选参数、默认参数、可变参数和关键字参数 func(*args, **kw)
  • list: 生成器, yield
  • map, reduce, filter, sorted: 序列映射, 归约, 过滤, 排序
  • 函数参数及返回(闭包)
  • functools.partial: 创建新函数固定住原函数的部分参数, 调用时更简单.
  • __slots__ @property: 限制可动态设置的属性, 简化属性调用

4 python pep8

转载自http://www.douban.com/note/134971609/

  1. 编码中考虑到其他python实现的效率等问题，比如运算符‘+’在CPython（Python）中效率很高，都是Jython中却非常低，所以应该采用.join()的方式。
  2. 与No单件比较，尽可能使用‘is’‘is not’，绝对不要使用‘==’，比如if x is not None 要优于if x。
  3. 使用startswith() and endswith()代替切片进行序列前缀或后缀的检查。比如：建议使用if foo.startswith('bar'): 而非if foo[:3] == 'bar':
  4. 使用isinstance()比较对象的类型。比如：建议使用if isinstance(obj, int): 而非if type(obj) is type(1):
  5. 判断序列空或不空，有如下规则：建议使用if [not] seq: 而非if [not] len(seq)
  6. 二进制数据判断使用 if boolvalue的方式。

5 pdb调试

b 行号，函数名
设置断点, 不带参数显示所有断点
condition 行号 条件
条件断点
cl 行号
清除断点
j 行号
跳转到指定行执行, 只在 call stack 最底层起作用
! 语句
修改变量
a
打印当前函数参数(栈内变量)
w d u
列出 call stack, 下移一层, 上移一层
disable enable
取消, 恢复断点功能
ignore 次数
忽略断点功能一定次数

7 打包

7.1 py2exe

一般调用方法: python setup.py py2exe, 其中setup.py如下:

from distutils.core import setup
import py2exe
import sys
sys.argv.append("py2exe")
includes = ["encodings", "encodings.*"]
options = {"py2exe":
           { "compressed" : 1,
             "optimize" : 2,
             "bundle_files" : 2,
             "includes" : INCLUDES,
             "dll_excludes": [ "MSVCP90.dll", "mswsock.dll", "powrprof.dll","w9xpopen.exe"] }}
setup(options = options,
      description = "车位管理系统",
      zipfile=None,
      console = [{"script":'TaskNameList.py', 'icon_resources':[(1, 'logo.ico')]}])

指定额外的文件

# mysetup.py
from distutils.core import setup
import glob
import py2exe

setup(console=["helloworld.py"],
data_files=[("bitmaps",
              ["bm/large.gif", "bm/small.gif"]),
             ("fonts",
              glob.glob("fonts\\*.fnt"))],
)

如上配置, 也可这样调用 python setup.py
