3 oracle 基础

3.1 oracle 用户和权限

  • 视图 dba_users 存储所有用户的基本信息，如下查看

    SQL> select username, account_status, default_tablespace, temporary_tablespace from dba_users;
  • 用户在数据库中所拥有的所有对象的集合（表、视图、存储过程、触发器）
  • 系统用户
    • sys 角色为 sysdba，管理员，最高权限
    • system 角色为 sysoper，操作员，权限次之
  • 权限
    • 系统权限
    • 对象权限
  • 权限的集合，便于更灵活的分配权限

4 一些具体问题

4.1 listener 服务无法启动，导致无法连接数据库

通常是改变了主机名导致，解决办法是：

  1. 打开 Net manager
  2. 监听程序-listener-主机 改成对应名字，一般改成 localhost 最好
  3. 启动 listener 服务

对于 2，也可删除 listener 再增加一个，程序会根据当前主机名重新建立。
