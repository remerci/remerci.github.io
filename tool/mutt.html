1 快捷键

<example>
@ 显示发件人邮箱
$ 保存邮箱更改，如删除标记删除的邮件
HML 到 index 的上、中、下
zZ index 上下翻页
*= 移动到 index 的顶和底
F,W 邮件重要标记；邮件所有标记
ST skip or display quoted text
e (pager) edit original mail
^E 改附件类型
^ 移动到 pager 顶
\ (pager) toggle search coloring

,! pipe message to command; shell command

esc e resend message
</example>

2 用 lbdb 地址管理

apt-get 安装 little brother 数据库，在 mutt 中按 Q 进行地址查询

3 html 邮件乱码

一些 html 邮件不写明白编码，这样直接用 firefox 和 w3m 都只能看到乱码。
解决办法是.mailcap 中这样来写

<example>
text/html; w3m -o document_charse=%{charset} %s; nametemplate=%s.html
text/html; w3m -dump -o document_charse=%{charset} %s; nametemplate=%s.html; copiousoutput
</example>

4 终端图片查看

一些邮件的 html 附件用的图片用 firefox 等无法看到，而附件查看方式默认用 imagemagick 查看，比较麻烦。参考网上的方法用 w3m 来看。

apt-get 下载 w3m-img 包，.mailcap 中这样写
<example>
image/*; ~/.mutt/w3m-img %s
</example>
其中，w3m-img 是脚本，内容如下：
<example>
#!/bin/sh
echo \<img src="\"$1\"" \\\> > ${1}.html
w3m ${1}.html
rm ${1}.html
</example>
