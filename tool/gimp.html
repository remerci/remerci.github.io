最近感到写文章、作报告时需要的一些图片，在 linux 上没有什么特别好的工具。
尤其是 mspaint 这样简单强大的小工具，还有集成很多图库的 ms visio。
想到自己可能对 linux 下的画图工具还不熟悉，像 metapost，inkspace，gimp，xfig，
dia，imageMagick 等等。
就想先好好学习一下，这里是 gimp 的一些学习心得。

开始读的是 gimp 自己带的文档，居然还是中文的，不过光讲解工具感觉收获不大。
于是网上找到两个资源。官方的 tutorial，以及这个网站。

这两天看完了官方 tutorial 的 for beginner 的部分，列一下主要的收获。

  • 快速蒙板的使用，配合过滤器，生成各式边框特效。
  • 利用通道和选择工具，可以用笔刷画图。
  • 通道的点可以操作以作出曲线效果。
  • 主要收获：利用多个图层和图层蒙板作出带有阴影效果的文字 logo。

1 gimp 对照片的处理

  • 消除红眼

    选中眼睛区域, 菜单 filter -> enhance -> red eye removal.

  • 照片亮度太暗

    复制图层, 对该图层操作: 图层对话框的模式中选择 screen.

2 生成透明背景

在背景区域右键选 图层 -> 透明度 -> 将色彩转为透明, 对应英文菜单是
layer -> transparency -> color to alpha. 还可进一步设置 transparency 中的
threshold alpha.
